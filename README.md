# SolarMap AI ☀️

SolarMap AI is a web platform that analyzes the solar installation potential of a location and helps users make smarter solar investment decisions.

>Location → Site Inputs → Solar Analysis → Energy Forecast → Savings Estimate → Downloadable Report

Users can enter their site details, run a solar analysis, and receive insights such as solar suitability score, recommended panel type, energy generation estimates, and financial savings projections.

## ✨ Features

🔐 User authentication (Email + Google login)

📍 Smart location input
• Detect current location
• Address search with autocomplete
• Automatic latitude & longitude detection

☀️ Solar potential analysis
• Roof area input
• Obstruction level selection
• Energy usage estimation
• Budget estimation

📊 Dashboard
• Run new solar analyses
• View analysis history
• Access generated results

📄 Downloadable solar report (PDF)

🌐 Multi-language support
• English
• Hindi
• Marathi

🎨 Modern responsive UI with dark/light mode
